结构

调用：以下举例常用接口，根据需要使用

       引入： 
         $(document).ready(function () {
            $(".use97").focus(function () {
                WdatePicker({
                    dateFmt:'您选择的日期是:yyyy年MM月dd日 HH时mm分', //格式配置
                    skin:'zxjorange',                                                   //皮肤
                    
                    isShowClear:false,		                                //显示"清除"
                    firstDayOfWeek:1,                                           //周的第一天（周：从周一开始显示）
                    minDate:"2016-04-26 00:00:00",                          //可用的起始时间
                     maxDate:"2017-09-30 00:00:00",                          //可用的结束时间
                    autoPickDate:true,                                      //true，选定日期即可自动输入
                    lang:"zh-cn1",      //WdatePicker.js     1.$langList 增加     {name:"zh-cn1",charset:"UTF-8"},
                                        //2.lang目录下增加zh-cn1.js
                    onpicked:function(dp){                                  //选中回调
                        var val=dp.cal.getDateStr();                        //取值
                    }
                })
            });

        })

更改皮肤方法

    1.找到My97DatePicker\WdatePicker.js

    2.$skinList数组中增加配置 {name:"皮肤名",charset:"gb2312"}

    3.My97DatePicker\skin 下新建相关皮肤文件夹    eg: zxjorange(建议拷贝样式及图片做更改)

        

    4.调用时skin:"皮肤名"
