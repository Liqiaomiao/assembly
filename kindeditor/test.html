效果

说明

            如需更改表情
            1. kindeditor/plugins增加文件夹my(否则报错)，将图片、样式、js放入my文件夹
            2.引入
             <script src="kindeditor.js"></script>
             <script src="lang/zh_CN.js"></script>
             <script src="plugins/my/emoticons.js"></script>
             <link rel="stylesheet" type="text/css" href="plugins/my/my.css" >

            不需更改表情，只需引入
            <script src="kindeditor.js"></script>
            <script src="lang/zh_CN.js"></script>

结构

调用

        KindEditor.ready(function (K) {
               $editor =   K.create('#test',{
                    width:"590px",
                    height:"80px",
                    items:['emoticons']

            });
            //赋值
            $editor.html("123");

         });
        //取值
        $("input").click(function () {
            alert($editor.html())
        })
